## Next steps

1. **Push this project** to your repo. In this folder, run:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Paws & Preferences"
   git branch -M main
   git remote add origin https://github.com/arifakbarr/paws-and-preferences.git
   git push -u origin main
   ```
   

2. **Turn on GitHub Pages** — In the repo: **Settings → Pages**. Under **Build and deployment**, set **Source** to **GitHub Actions**. The workflow will run on push; your site will be at `https://arifakbarr.github.io/paws-and-preferences/`.

3. **You’re set.** The links above are ready to submit.

Then submit the two links from the top of this file.
